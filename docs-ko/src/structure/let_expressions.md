# `let`식

Par의 여러 가지 타입과 식을 둘러보기 전에 마지막으로 알아야 할 것이 있다. `let`식을 사용해 변수를 정의하고 다른 식에서 사용할 수 있다.

키워드 `let`으로 시작해서 소문자 변수 이름, 등호, 변수에 대입할 값, 그리고 키워드 `in` 다음에 마지막으로 그 변수를 사용하는 식을 작성하면 된다.

글로 적으니 다소 복잡하다.

```par
module Main

import @core/Nat

def Six = let three = 3 in three + three
```

등호의 좌변에는 단순히 변수명만 올 수 있는 것이 아니다!

우선 타입 표기를 추가할 수 있고...

```par
def Six = let three: Nat = 3 in three + three
```

변수 대신 패턴을 작성할 수도 있다.

```par
def Twelve = let (a, b)! = (3, 4)! in a * b
```

위의 패턴은 [순서쌍](../types/pair.md) 패턴과 [단위](../types/unit.md) 패턴을 조합한 것이다. 이 두 타입에 대해서는 곧 다루겠다.

> 타입 표기는 반드시 변수명 바로 뒤에 온다. 즉, 이렇게 쓰면 안 된다.
>
> ```par
> let (a, b)! : (Nat, Nat)! = (3, 4)! in ...  // 오류!
> ```
>
> 타입 표기가 변수명 바로 뒤에 오지 않기 때문이다. 아래와 같이 쓰는 것은 괜찮다.
>
> ```par
> let (a: Nat, b: Nat)! = (3, 4)! in ...      // 올바른 코드
> ```

**이제 정말로 여러 가지 타입과 식을 둘러보자!**
